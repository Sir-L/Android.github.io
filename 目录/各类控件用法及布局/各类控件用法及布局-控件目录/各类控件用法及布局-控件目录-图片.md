# Android笔记

​				——各类控件的使用（控件）-图片

#### 图片：ImageView

##### 属性：

|    XML属性    |   说明   |
| :-----------: | :------: |
| layout_width  | 布局宽度 |
| layout_height | 布局高度 |
|      src      | 图片内容 |
|  foreground   |   前景   |
|  background   |   背景   |

##### 方法：

|      常用方法      |                             说明                             |
| :----------------: | :----------------------------------------------------------: |
|  setVisibility()   | 是否可见（View.VISIBLE 可见；View.INVISIBLE 不可见，但依然占据位置；View.GONE不可见，且不占据位置） |
| setImageResource() |                           设置图片                           |
|   setImageAlpha    |                        设置图片透明度                        |
